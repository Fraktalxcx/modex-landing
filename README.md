# Modex Landing (GitHub Pages)

Статический лендинг для публикации через GitHub Pages.

## Быстрый запуск

1. Откройте `index.html` локально и проверьте внешний вид.
2. Для формы замените в `index.html`:
   - `action="https://formspree.io/f/yourFormId"`
   - на ваш endpoint из Formspree.
3. Загрузите содержимое папки `docs/` в репозиторий.

## GitHub Pages

- Репозиторий -> Settings -> Pages
- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/docs`

После этого сайт будет доступен по URL GitHub Pages.
